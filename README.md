# Workspace Projects Overview

This workspace contains several Natural Language Processing (NLP) and data analysis projects. Below is a brief overview of each project:

## 1. Fake News Detection (`facknews`)
- **Purpose:** Detects fake news using machine learning models.
- **Key Files:**
  - `app.py`: Main application script.
  - `fack_news.ipynb`: Jupyter notebook for data analysis and model training.
  - `facknews_data/`: Contains datasets (`Fake.csv`, `True.csv`).
  - `fake_news_model/`: Pre-trained model files and configuration.
- **Description:** This project uses NLP techniques to classify news articles as fake or true. It includes data preprocessing, model training, and prediction.

## 2. Searching App (`searching_app`)
- **Purpose:** Provides search functionality, possibly for documents or datasets.
- **Key Files:**
  - `app.py`: Main application script.
  - `env/`: Python virtual environment with dependencies.
- **Description:** This app implements search features, likely using NLP or keyword matching to retrieve relevant information from provided data sources.

## 3. Sentiment Analysis (`sentment_analysis`)
- **Purpose:** Analyzes the sentiment of text data (e.g., reviews).
- **Key Files:**
  - `app.py`: Main application script.
  - `sentment_analysis.ipynb`: Jupyter notebook for sentiment analysis.
  - `DisneylandReviews.csv`: Dataset of reviews.
- **Description:** This project processes review data to determine sentiment (positive, negative, neutral) using NLP models.

## 4. Spam Detection (`spam_or_not`)
- **Purpose:** Classifies messages as spam or not spam.
- **Key Files:**
  - `app.py`: Main application script.
  - `spam_or_notspam.ipynb`: Jupyter notebook for spam detection.
  - `spam_or_not_spam.csv`: Dataset of messages.
- **Description:** This project uses machine learning to identify spam messages based on their content.

---

Each project includes code, datasets, and (where applicable) pre-trained models or notebooks for experimentation and further development.
